Choose Your Path

Complete the labs in order for a progressive introduction, or select the scenario that matches your current work. Portal labs focus on workflow design and operations. VS Code labs use Logic Apps Standard because local development is supported for Standard projects.

Before you begin: use a non-production Azure subscription, choose one region for related resources, and remove the lab resource group when you finish.

Common Prerequisites

  • An Azure subscription where you can create resource groups and Logic Apps resources.
  • For portal labs: access to the Azure portal and permission to create API connections.
  • For VS Code labs: Visual Studio Code, the Azure Logic Apps (Standard) extension, and Azure Functions Core Tools.
  • A unique suffix for globally scoped resource names, such as your initials plus four digits.

Security

Prefer managed identity and least-privilege role assignments. Do not place credentials, connection strings, or access keys in workflow definitions or source control.
